Política de Privacidad

Sandy Wood creó la aplicación Sissy Academy Conditioning Helper como una aplicación de código abierto. Este SERVICIO es proporcionado por Sandy Wood sin costo y está diseñado para usarse tal como está.

Academia Sissy ha adaptado la aplicación y sigue las mismas políticas de privacidad que el autor original.

Esta página informa a los visitantes sobre las políticas con respecto a la recopilación, uso y divulgación de información personal.

Si decide usar el Servicio, acepta la recopilación y uso de información de acuerdo con esta política. La información personal se utiliza para proporcionar y mejorar el Servicio y no se compartirá salvo lo descrito aquí.

Recopilación y uso de información

Para una mejor experiencia, es posible que se solicite cierta información personal, incluyendo:

  • Metadatos de archivos (nombres, tamaño) de Google Drive.
  • La información se almacena en su dispositivo y no se recopila ni comparte.
  • Servicios de terceros pueden recopilar información para identificarle.

Datos de registro

En caso de error, la aplicación recopila datos de registro, incluyendo dirección IP, nombre del dispositivo, versión del sistema operativo, configuración de la aplicación, fecha y hora de uso, y otras estadísticas.

Cookies

Las cookies son pequeños archivos usados como identificadores únicos. Este servicio no utiliza cookies explícitamente, pero puede usar bibliotecas de terceros que sí. Usted puede aceptarlas o rechazarlas.

Proveedores de Servicios

Puedo contratar empresas o personas externas para:

  • Facilitar o prestar el Servicio.
  • Realizar servicios relacionados o analizar el uso del Servicio.

Estos terceros tienen acceso a su información personal pero no pueden usarla para otros fines.

Servicios de terceros incluyen: Google Drive API, GitHub Pages, Seguridad.

Enlaces a otros sitios

Este servicio puede contener enlaces a otros sitios. Al hacer clic, será redirigido a ellos. Revise sus políticas de privacidad, ya que no tengo control sobre ellos.

Privacidad infantil

Este servicio no está dirigido a menores de 18 años. No se recopila información de menores de 13 años. Si se detecta que un menor ha proporcionado información, será eliminada inmediatamente.

Cambios a esta Política de Privacidad

Podemos actualizar la política ocasionalmente. Se recomienda revisar esta página periódicamente para estar al tanto de cambios. Los cambios entran en vigor al publicarse.

Contáctenos

Si tiene preguntas o sugerencias sobre esta Política de Privacidad, no dude en ponerse en contacto con nosotros.
